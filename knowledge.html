RSA

Generate 2 large primes \(p\) , \(q\)
\(n=pq,\ \phi(n)=(p-1)(q-1)\)
Choose random \(b\) that \(1<b<\phi(n),\ gcd(b,\ \phi(n))=1\)
\(a\equiv b^{-1}\ mod\ \phi(n)\)
Encrypt \(y\equiv x^b\ mod\ n,\) decrypt \(x\equiv y^a\ mod\ n\)

We need \(p\) and \(q\) to be at least 512 bits long.
Generate large primes = Apply primality test to large numbers
Primality test algorithms: Solovay-Strassen, Miller-Rabin

Solovay-Strassen

Quadratic Residue
Given a prime \(p > 2,\) for \(1\leq x\leq p-1,\) if there exists a \(y\) that \(y^2\equiv x(mod\ p)\),
then x is defined to be a quadratic residue modulo p; otherwise it's called a quadratic non-residue.

Legendre Symbol
Given a prime \(p > 2,\) for any integer \(a\geq 0,\) define the Legendre symbol as \[ \left(\frac{a}{p}\right)=\left\{ \begin{aligned} 1 \ ,&\ a\ is\ a\ quadratic\ residue\ modulo\ p \\ 0 \ ,&\ p\ devides\ a \\ -1\ ,&\ a\ is\ a\ quadratic\ non{\text -}residue\ modulo\ p \end{aligned}\right. \]
